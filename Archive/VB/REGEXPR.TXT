RegExpr for Visual Basic 5.0 (and later)

*** DEMO ***

RegExpr is a regular expression engine for Visual Basic 5.0 and later.
This is a demo EXE showing the features of the engine. The real product
is RegExpr.Bas, an ordinary VB module that can be included in any VB
project. 

System requirements: 
* Windows 95/NT
* Visual Basic 5.0 or the run-time library
