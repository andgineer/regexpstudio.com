Installation and file list


To install the library copy RegExpr.pas in your project directory or simply add path to it into Your project manager.

That's all !

Now just use TRegExpr object or global routines in Your project (see simple illustrations). Don't forget to add 'uses RegExpr' into appropriate units of Your project.
