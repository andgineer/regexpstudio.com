Demo project (TestRExp)


Simple program for r.e. exploring and testing, distributed in sources (project TestRExp.dpr) and as TestRExp.exe.

Note, that it uses some VCL properties which exists only in Delphi 4 or higher. While compiling in Delphi 3 or Delphi 2 you'll recieve some error messages about unknown properties. You may ignore it - this properties is needed only for resizing and justification of components then form change it's size.

With help of this proggy, You can easely determine number of subexpression You are editing, jump to any subexpression defined (in r.e. source code as well as in search results in input string), play with Substitude, Replace and Split functions and so on.

Moreover, into the demo-project included a lot of examples - use it while learning r.e. syntax or to quick exploring TRegExpr features.
