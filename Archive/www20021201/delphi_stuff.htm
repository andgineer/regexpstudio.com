Regular expressions for Delphi !  

TRegExpr v.0.951

Easy to use and powerfull tool for sophisticated search and substitutioning and for template-based text checking (especially usefull for user input validation in DBMS and web projects).

You can validate e-mail adresses, extract phone numbers or ZIP-codes from web-pages or documents, search for complex patterns in log files and all You can imagine! Rules (templates) can be changed without Your program recompilation!

As a language for rules used subset of Perl's regular expressions (regexp).

Full source codes included, pure Object Pascal. Thus, You need no DLL!

Audio Mpeg(mp3) frame headers, ID3v1 for Delphi !  

TAudioInfo

Freeware (with full source) Delphi library for fast extracting mp3 (MPEG 1, 2 Layers I, II, III) headers and for working with ID3v1.1 tags (read/write).
Supported Xing VBR (variable bitrate) format !
Demo project included.

Download audioinf.zip (14K)

You can find detailed description of mpeg headers and ID3v1.1 (sorry, only russian. Some information in English you can take from Gustav Munkby page and www.id3.org) !
