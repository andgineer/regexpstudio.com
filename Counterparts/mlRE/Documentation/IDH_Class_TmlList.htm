TmlList Object
Properties Methods

Unit
mlObject

Declaration
TmlList = class(TObject)

Description
TList style object to contain a list of TmlObjects and handle appropriate reference counting. When an object is added to the list, it's reference count is incremented and when removed decremented.

Introduced Public Properties
Capacity  Capacity of list
Count  Count of Interfaces in the list
Objects  Access to the interfaces in the list
