GetTransitionsOn method

Applies to
TmlRegularExpressionState

Declaration
Procedure GetTransitionsOn(cChar: char; oStateList: TList);

Description
Get all the transitions we can make on this character. If this state is the start of any lambda transitions, we implicitly follow them to find any additional destinations which are possible for this character. oTransitionList isnot cleared prior to adding our transitions.

Parameters
cChar  Character of input to recognise
oTransitionList  List to receive possible transitions
