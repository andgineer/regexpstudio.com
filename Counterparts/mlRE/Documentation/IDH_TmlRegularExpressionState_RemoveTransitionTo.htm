RemoveTransitionTo method

Applies to
TmlRegularExpressionState

Declaration
Procedure RemoveTransitionTo(oState: TmlRegularExpressionState; xCharacters: TCharset);

Description
Remove transition to the passed state from this state on these characters. If there are no other characters left, the old transition will be removed.

Parameters
oState  Destination state
xCharacters  Characters to remove from the transition
