AddTransition method

Applies to
TmlRegularExpression

Declaration
Function AddTransition(oNewState: TmlRegularExpressionState; xCharacters: TCharSet): TmlRegularExpressionState;

Description
Add a transition to a state and set that new state as the current state.

Parameters
oNewState  State to transition to. If nil, creates a new ordinary state to use.
xCharacters  Set of characters to transition upon

Returns
New value of FoCurrentState
