TRegularExpressionParserFlag Type

Unit
mlRegularExpression

Declaration

TRegularExpressionParserFlag = (pfEscaped);

Description
A list of possible flags to further identify the nature of a parsed character.

Values
pfEscaped  Character was preceded by '\' in the input stream.

If the character normally has a special meaning (eg: ? * [ ] ) then it will be treated simply as a normal character with no special significance.
If the character normally has no significance, it may be treated with special significance (eg: \d = [0-9], \t = tab and so on)
