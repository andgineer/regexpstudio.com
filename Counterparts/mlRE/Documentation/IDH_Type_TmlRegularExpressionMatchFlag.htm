TmlRegularExpressionMatchFlag Type

Unit
mlRegularExpression

Declaration

TmlRegularExpressionMatchFlag = ( mfCaseInsensitive, mfLongestMatch, mfAllMatches, mfStartOnly,
                                  mfFinishOnly, mfOverlapMatches);

Description
Possible flags governing Regular Expression Matching.

Combining mfAllMatches with mvOverlapMatches may result in a large number of results.
Combinging mfStartOnly with mfFinishOnly requires that the entire line match the expression.

mfStartOnly can be specified by begining the Regular Expression with '^'
mfFinishOnly can be specified by ending the Regular Expression with '$'


Values
mfCaseInsensitive  Matches don't depend on case
(default case is case sensitive)
mfLongestMatch  Find the longest possible match to the RE
(default is to find the shortest)
mfAllMatches  Find all matches
(default is to find only the first)
mfStartOnly  Only match at the start of the string
(default is to find a match anywhere)
mfFinishOnly  Match must terminate at the end of the string
(default is to permit a match finishing anywhere)
mfOverlapMatches  Allow overlapping matches to be returned
(default is to return matches which do not overlap
