  TRegExpr library    

TRegExpr - Freeware Регулярные Выражения для Delphi (Regular Expressions)

Универсальный инструмент для проверки пользовательского ввода на соответствие шаблонам, для самого изощренного поиска, а также замены подстрок.
Убедитесь сами, насколько возрастает производительность если оторваться от традиционного процедурного подхода и посмотреть на текст с высоты птичьего полета! Free Your mind (c) Matrix 8-)
Вы можете проверить синтаксическую корректность e-mail адреса, извлечь номера телефонов из неформализованного текста, найти необходимую информацию в web-странице - возможности ограничены только Вашим воображением. Правила (шаблоны) можно изменять не перекомпилируя Вашу программу !
  TAudioInfo library    

TAudioInfo - заголовки Audio Mpeg(mp3), ID3v1 для Delphi

Freeware (с исходными текстами) Delphi библиотека для быстрого извлечения mp3 (MPEG 1, 2 Layers I, II, III) заголовков и для работы с тэгами ID3v1.1 (чтение и запись).
Поддерживается формат Xing VBR (сжатие с переменным bitrate) !
Имеется демо-программка.
