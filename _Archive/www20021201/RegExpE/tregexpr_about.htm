About

TRegExpr is easy to use and powerfull tool for template-based checking of user input string fields (in DBMS and web-based applications), text search/substitutions, egrep-like utils & etc.

You can easily check e-mail adress syntax, extract phone number or ZIP-code from unformatted text or any necessary information from web-pages and all You can imagine! Rules (templates) can be changed without program recompilation !

This freeware library is a extended Delphi-port of Henry Spencer V8-routins for working with subset of Perl Regular Expressions.

TRegExpr is written in plain object pascal with full source code avalable for free.

The original C source was improved and encapsulated completely into class TRegExpr implemented as a single file: RegExpr.pas.

Thus, You need no DLL!

Take a look at simple illustrations and study regular expression syntax (You can use demo project for exploring and debugging Your own regular expressions).

You can use Unicode (Delphi's WideString) - see "How to use unicode".

See What's New section for latest changes.
