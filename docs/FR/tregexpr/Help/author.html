Cette librairie est dérivé des fichiers sources de Henry Spencer.
J'ai transféré les fichiers source en objet pascal, et y ai ajouté d'autres fonctions.
Plusieurs de ces nouvelles fonctions ont été faites à partir de suggestion ou à partir de modification partielle d'autre utilisateurs de TRegExpr (voir la liste de gratitude plus bas).
