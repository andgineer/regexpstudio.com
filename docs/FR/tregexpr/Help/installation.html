Installation et liste de fichier

Pour installer la copie de la librairie installer RegExpr.pas dans le répertoire de votre projet ou tout simplement l'ajouter au répertoire de votre projet dans votre manageur de projet. C'est aussi simple que cela.

Maintenant utiliser les objets ou les routines de TRegExpr dans votre projet (voir les simples illustrations). Ne pas oublier d'ajouter l'usage de l'unité de 'RegExpr' dans votre projet.

Fichiers d'aide (distribués séparément dans quelques versions de TRegExpr)
·    RegExpE.hlp, RegExpE.cnt - Aide Anglaise  
·    RegExpRu.hlp, RegExpRu.cnt - Aide Russe  
·    RegExpBg.hlp, RegExpBg.cnt - Aide Bulgare (par Simeon Lilov)  
·    RegExpG.hlp, RegExpG.cnt - Aide Allemande (par Martin Baur)  
·       RegExpF.hlp, RegExpF.cnt - Aide en Français (par Martin Ledoux)
·    emplacement pour l'aide dans votre langue ;) SVP, m'aider pour la traduction !  

Projet Demo consiste en ces fichiers:
·    TestRExp.dpr (fichier de projet).  
·    TestRE.pas (forme principale).  
·    TestRE.dfm  
·    PCode.pas (visionneuse de pseudocode (compilation de l'e.r.)).  
·    PCode.dfm  
·    FileViewer.pas (visionneuse pour la recherche dans un fichier comme egrep).  
·    FileViewer.dfm  
·    HyperLinkDecorator.pas (exemples plus complexe).  
·    test.htm (fichier demo pour la visionneuse de FileViewer).
