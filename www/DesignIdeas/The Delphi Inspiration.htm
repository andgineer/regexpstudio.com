Containers

DIContainers is a collection of more than 60 distinct container classes for Borland Delphi.

The containers are divided into 4 different structures: Hash containers, doulbly-linked Lists, N-ary Trees, and Vector containers.

Containers are readily available for AnsiStrings, WideStrings, different Number types, Objects, and various combinations of the above. Advanced container operations include cross-container assignment and cascading streaming. More...


FileFinder

DIFileFinder is a Delphi Component to search the file system for files and or folders matching (or not matching!) particular attributes and / or name patterns. Pattern syntax includes standard wildcards as well as full regular expressions.

DIFileFinder suports inclusion as well as exclusion filters. Unlike most file finder components, DIFileFinder does not use events nor does it build lengthy file lists. Instead it reports a single file or folder at a time. More...


HtmlParser

DIHtmlParser is the only fully Unicode / WideString HTML Parser for Borland Delphi. It can read from and write to more than 40 character sets natively.

DIHtmlParser is the perfect development tool to quickly extract information from HTML documents, modify existing create new HTML documents.

The core TDIHtmlParser component recognizes 14 different pieces of HTML and Non-HTML, like HTML-Tags, text, comments, scripts, styles, titles, Custom-Tags, PHP, and others.

Plugins are available to extend the HTML Parser's functionality. They allow to extract additional information "on the fly" next to the main parsing process. More...


Pcre - Regular Expressions

DIPcre is the most advanced regular expression engine for Boland Delphi.

The implemented regular expression syntax is Perl 5 compatible and provides full substring capturing, case sensitive and insensitive matching, greedy and non-greedy repetition quantifiers, backtracking, assertions and more.

DIPcre also includes two regular expression edits: TDIPcreEdit and TDIPcreComboBox. Both validate user entry against a regular expression pattern and are quite similar to, but more powerfull then Delphi's TMaskEdit. More...


ZipWriter

DIZipWriter is a Delphi component to create PKZip-compatible .zip archives. New .zip archives may be created as files or as streams.

The TDIZipWriter class has plenty of methods to write various types of data to the archive, including Files, Streams, Strings, and Buffers. Multiple writes to the same archive entry are possible. Direct streaming allows objects to stream their data directly to a TStream interface using their native WriteToStream procedure. More...


Mime Encoder & Decoder

DIMime implements the fastest Mime Encoding & Decoding currently available for Borland Delphi.

DIMime has and efficient interface for Buffers, Strings and Streams and supports encoding with or without linebraks for all input types.

DIMime is written in speed optimized Delphi Pascal and is part of the Jedi Code Library (JCL). More...


PasDoc

PasDoc generates HTML and HTMLHelp documentation from comments in Pascal unit source code files.

It is the only free help generator that flawlessly follows compiler defines and include files. More...
